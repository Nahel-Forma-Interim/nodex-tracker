Comment ça marche

Format attendu : NDX-{TIER}-XXXX-XXXX-XXXX où TIER ∈ ESS (Essentiel), PRO, ELT (Elite), DEMO ou OWNER.

Validation locale (mode actuel) :

  • Étape 1 : la regex format. Si raté → rejet immédiat.
  • Étape 2 : check des codes spéciaux (DEMO/OWNER). Toujours valides.
  • Étape 3 : tout autre code au bon format est accepté (en attendant le backend).

Tu as droit à 3 essais. Au 3e échec, le skill bloque pendant 24h (anti-brute-force).

Mode serveur (à venir) : appel à nodextrading.com/api/verify?code=... qui vérifie en base que le code existe, n'est pas révoqué, et appartient à l'email de l'acheteur.
